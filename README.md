To install packages, use the command npm install

## Commands

### Starting the development server
```shell
npm run start
```

### Project collection without optimization
```shell
npm run build-dev
```

### Project collection with optimization
```shell
npm run build-prod
```

### Clearing the dist folder
```shell
npm run clear
```
